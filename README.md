# Pet Diary
Web Application for pet

# How to start
## Backend
### Command
`default root`
`./mvnw spring-boot:run`
`https://localhost:8080`

### Spec
- Spring boot (maven)
- PostgreSQL (port: 5462)
- JAVA

## Frontend
### Command
`/petdiary/src/main/frontend`
- `yarn dev` `https://localhost:3000`
- `yarn generate`
- `yarn lint`
- `yarn storybook`

### Component
- atom
- molecure
- organism

### Style
- scss

### Spec
- Next.js
- Sass
